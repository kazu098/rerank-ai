## 推奨設定

### MVP・開発環境（無料枠活用）
```env
LLM_PROVIDER=qwen
QWEN_API_KEY=your-qwen-api-key
```
- **理由**: 無料枠が明確（1,000回/日）、コストが予測しやすい
- **コスト**: 約0円（無料枠内）

### 本番環境（スケール時）
```env
LLM_PROVIDER=groq
GROQ_API_KEY=your-groq-api-key
```
- **理由**: 高速、無料枠が大きい、日本語対応良好
- **コスト**: 約0.30-0.68円/記事

### 高品質が必要な場合
```env
LLM_PROVIDER=gemini
GEMINI_API_KEY=your-gemini-api-key
# または
LLM_PROVIDER=openai
OPENAI_API_KEY=your-openai-api-key
```
- **理由**: 高精度な分析結果が必要な場合
- **コスト**: 約0.14-3.38円/記事（モデルによる）

## コスト最適化のポイント

1. **無料枠の活用**: Qwen、Groq、Geminiの無料枠を最大限活用
2. **モデルの選択**: 用途に応じて最適なモデルを選択（Flash Lite vs Pro）
3. **出力トークンの削減**: プロンプトを最適化して出力トークンを削減
4. **バッチ処理**: 複数の記事をまとめて処理して効率化
5. **キャッシュ**: 同じ記事の再分析を避ける
