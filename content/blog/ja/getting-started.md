# ReRank AIの使い方：初心者向け完全ガイド

ReRank AIは、Google Search Consoleと連携して、検索順位の下落を自動検知し、AIが改善案を提案するSEO自動化ツールです。このガイドでは、初めて使う方向けに、基本的な使い方を詳しく解説します。

## 1. アカウント作成

まず、ReRank AIにアカウントを作成しましょう。

1. [ReRank AIのトップページ](https://rerank-ai.com)にアクセス
2. 「今すぐ始める」ボタンをクリック
3. メールアドレスまたはGoogleアカウントで登録

登録は無料で、すぐに始められます。

## 2. Google Search Consoleとの連携

ReRank AIの最大の特徴は、Google Search Console（GSC）と連携して、自動的に順位を監視することです。

### 連携手順

1. ダッシュボードの「サイトを追加」をクリック
2. 「Google Search Consoleと連携」を選択
3. Googleアカウントで認証（OAuth 2.0）
4. 監視したいサイトを選択

これだけで、自動監視が開始されます。

## 3. 記事の登録

監視したい記事を登録しましょう。

1. ダッシュボードの「記事を追加」をクリック
2. 記事のURLを入力
3. 監視したいキーワードを入力（複数可）

登録した記事は、毎日自動的に順位をチェックされます。

## 4. 通知設定

順位が下落した際に、通知を受け取る設定を行います。

### メール通知

1. 設定ページの「通知設定」を開く
2. 「メール通知」を有効化
3. 通知を受け取るメールアドレスを確認

### Slack通知（オプション）

チームで共有したい場合は、Slack通知も設定できます。

1. 「Slack連携」をクリック
2. Slackアプリをインストール
3. 通知を受け取るチャンネルまたはDMを選択

## 5. 分析結果の確認

順位が下落した場合、ReRank AIが自動的に分析を行い、改善案を提示します。

1. ダッシュボードで通知を確認
2. 「改善案を見る」をクリック
3. AIが提案した改善項目を確認
4. 記事を修正して反映

## まとめ

ReRank AIを使えば、SEO対策が自動化され、手動で順位をチェックする必要がなくなります。まずは無料プランで試してみて、効果を実感してください。

## よくある質問

**Q: 無料プランで何記事まで監視できますか？**
A: 無料プランでは3記事まで監視可能です。

**Q: GSC連携は必須ですか？**
A: 自動監視を使う場合は必須です。手動でスキャンする場合は不要です。

**Q: 通知はどのくらいの頻度で届きますか？**
A: 順位が設定した閾値を下回った場合、即座に通知が届きます。
