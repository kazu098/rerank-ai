# GSC連携方法

ReRank AIの最大の特徴は、Google Search Console（GSC）と連携して、自動的に順位を監視することです。

## 連携手順

1. ダッシュボードの「サイトを追加」をクリック
2. 「Google Search Consoleと連携」を選択
3. Googleアカウントで認証（OAuth 2.0）
4. 監視したいサイトを選択

これだけで、自動監視が開始されます。

## 注意事項

- GSC連携には、Googleアカウントの認証が必要です
- 監視したいサイトのGSCにアクセス権限が必要です
- 連携後、初回の順位データ取得まで数時間かかる場合があります

## トラブルシューティング

### 連携エラーが発生する場合

- Googleアカウントの認証状態を確認
- ブラウザのCookieをクリアして再試行
- 別のブラウザで試す

### サイトが表示されない場合

- GSCでサイトが正しく登録されているか確認
- サイトの所有権確認が完了しているか確認
