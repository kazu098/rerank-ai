# はじめに

ReRank AIへようこそ！

ReRank AIは、Google Search Consoleと連携して、検索順位の下落を自動検知し、AIが改善案を提案するSEO自動化ツールです。

## ReRank AIでできること

- **自動順位監視**: Google Search Consoleと連携して、記事の順位を自動監視
- **順位下落の自動検知**: 順位が下がった際に、即座に通知
- **AI改善案提案**: 競合分析をもとに、具体的な改善案を自動生成
- **リライト原稿の自動生成**: 改善すべき項目を箇条書きで提示

## 主な機能

### 1. 手動スキャン

GSC連携が不要で、URLとキーワードを入力するだけで、現在の順位と改善案を即座に表示します。

### 2. 自動トリガー

GSC連携後、順位下落を検知し、自動で分析・通知します。

### 3. 競合との差分抽出

上位サイトとの違いを特定し、追加すべき項目を具体的に提示します。
