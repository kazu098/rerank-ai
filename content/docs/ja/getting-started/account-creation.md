# アカウント作成

ReRank AIのアカウント作成は、無料で簡単に行えます。

## アカウント作成手順

1. [ReRank AIのトップページ](https://rerank-ai.com)にアクセス
2. 「今すぐ始める」ボタンをクリック
3. 以下のいずれかの方法で登録：
   - **メールアドレス**: メールアドレスとパスワードを入力
   - **Googleアカウント**: Googleアカウントでログイン（推奨）

## 登録後の確認

アカウント作成後、確認メールが送信されます。メール内のリンクをクリックして、メールアドレスを確認してください。
