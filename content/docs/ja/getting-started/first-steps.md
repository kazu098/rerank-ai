# 最初のステップ

ReRank AIを始めるための最初のステップを説明します。

## 5分で始めるReRank AI

### ステップ1: アカウント作成（1分）

1. [ReRank AIのトップページ](https://rerank-ai.com)にアクセス
2. 「今すぐ始める」ボタンをクリック
3. メールアドレスまたはGoogleアカウントで登録

### ステップ2: GSC連携（2分）

1. ダッシュボードの「サイトを追加」をクリック
2. 「Google Search Consoleと連携」を選択
3. Googleアカウントで認証
4. 監視したいサイトを選択

### ステップ3: 記事の登録（1分）

1. 「記事を追加」をクリック
2. 監視したい記事のURLを入力
3. 監視したいキーワードを入力
4. 「登録」をクリック

### ステップ4: 通知設定（1分）

1. 設定ページの「通知設定」を開く
2. メール通知またはSlack通知を有効化
3. 通知条件を設定

## これで完了！

これで、ReRank AIの基本的な設定が完了しました。順位が下がった際に、自動的に通知が届き、改善案が提案されます。
