# 手動スキャンの使い方

手動スキャンは、GSC連携が不要で、URLとキーワードを入力するだけで、現在の順位と改善案を即座に表示する機能です。

## 使い方

1. ダッシュボードの「手動スキャン」をクリック
2. 分析したい記事のURLを入力
3. 監視したいキーワードを入力（複数可）
4. 「分析開始」をクリック

## 表示される情報

- **現在の順位**: 指定したキーワードでの検索順位
- **改善案**: 競合分析をもとにした改善項目の箇条書き
- **競合サイト**: 上位表示されている競合サイトのリスト

## 注意事項

- 手動スキャンは、リアルタイムの順位データを取得しますが、GSC連携と比べて精度が低い場合があります
- 自動監視を使う場合は、GSC連携が必要です
