# Slack通知の設定

ReRank AIは、SlackチャンネルまたはDMに通知を送信できます。チームで共有したい場合に便利です。

## 連携手順

1. 設定ページの「Slack連携」をクリック
2. 「Slackアプリをインストール」をクリック
3. 通知を受け取るワークスペースを選択
4. 通知を受け取るチャンネルまたはDMを選択

## 通知の種類

- **チャンネル通知**: 指定したチャンネルに通知を送信
- **DM通知**: 指定したユーザーのDMに通知を送信

## 通知内容

Slack通知には、以下の情報が含まれます。

- 順位が下がった記事のURL
- 順位が下がったキーワード
- 改善案の概要
- 詳細な分析結果へのリンク

## 注意事項

- Slack通知を使うには、Slackアプリのインストールが必要です
- チャンネル通知を使う場合は、チャンネルへの投稿権限が必要です
