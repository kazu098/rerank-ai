# 自動通知の設定

ReRank AIは、順位が下落した際に、自動的に通知を送信します。

## 通知設定

1. 設定ページの「通知設定」を開く
2. 通知を受け取る方法を選択：
   - **メール通知**: メールアドレスに通知を送信
   - **Slack通知**: SlackチャンネルまたはDMに通知を送信

## 通知条件の設定

- **順位下落の閾値**: 何位下がったら通知するか（デフォルト: 2位）
- **キーワード下落の閾値**: キーワードの順位が何位下がったら通知するか（デフォルト: 10位）
- **比較期間**: 何日前の順位と比較するか（デフォルト: 7日前）

## 通知内容

通知には、以下の情報が含まれます。

- 順位が下がった記事のURL
- 順位が下がったキーワード
- 改善案の概要
- 詳細な分析結果へのリンク
