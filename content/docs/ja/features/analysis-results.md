# 分析結果の確認方法

ReRank AIが生成した分析結果の確認方法を説明します。

## 分析結果の見方

1. ダッシュボードで通知を確認
2. 「改善案を見る」をクリック
3. 分析結果ページで詳細を確認

## 分析結果に含まれる情報

### 1. 現在の順位

指定したキーワードでの現在の検索順位が表示されます。

### 2. 順位の推移

過去7日間の順位の推移がグラフで表示されます。

### 3. 改善案

AIが分析した改善案が箇条書きで表示されます。

### 4. 競合サイトの分析

上位表示されている競合サイトの分析結果が表示されます。

## 改善案の活用方法

1. **優先順位をつける**: 重要度の高い改善案から対応
2. **段階的に改善**: 一度にすべてを改善せず、段階的に改善
3. **効果を測定**: 改善後の順位変化を確認
