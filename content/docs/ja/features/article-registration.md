# 記事の登録方法

ReRank AIで監視したい記事を登録する方法を説明します。

## 記事の登録手順

1. ダッシュボードの「記事を追加」をクリック
2. 監視したい記事のURLを入力
3. 監視したいキーワードを入力（複数可）
4. 「登録」をクリック

## キーワードの選び方

- **メインキーワード**: 記事の主要なキーワード
- **関連キーワード**: 記事に関連するキーワード
- **長尾キーワード**: より具体的なキーワード

## 注意事項

- 1記事あたり、最大10個のキーワードを登録できます
- 無料プランでは、3記事まで登録可能です
- 有料プランでは、プランに応じて登録可能な記事数が異なります
