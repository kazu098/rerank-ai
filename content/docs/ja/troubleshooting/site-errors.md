# サイト登録エラー

サイト登録時にエラーが発生した場合の対処法を説明します。

## よくあるエラー

### サイトが表示されない

**原因**: GSCでサイトが正しく登録されていない

**対処法**:
1. Google Search Consoleでサイトが正しく登録されているか確認
2. サイトの所有権確認が完了しているか確認
3. 別のGoogleアカウントで試す

### アクセス権限エラー

**原因**: GSCへのアクセス権限がない

**対処法**:
1. Google Search Consoleでサイトへのアクセス権限を確認
2. サイトの所有者またはユーザーとして追加されているか確認
3. 別のGoogleアカウントで試す

### 連携エラー

**原因**: GSC連携が正しく完了していない

**対処法**:
1. 設定ページの「GSC連携」で、連携状態を確認
2. 「再認証」をクリックして、再認証を試す
3. ブラウザのCookieをクリアして再試行
