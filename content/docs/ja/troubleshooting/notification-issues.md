# 通知が届かない

通知が届かない場合の対処法を説明します。

## 確認事項

### 1. 通知設定の確認

- 設定ページの「通知設定」で、通知が有効になっているか確認
- メール通知またはSlack通知が有効になっているか確認

### 2. 通知条件の確認

- 順位下落の閾値が適切に設定されているか確認
- 比較期間が適切に設定されているか確認

### 3. メールアドレスの確認

- 登録しているメールアドレスが正しいか確認
- 迷惑メールフォルダを確認

### 4. Slack連携の確認

- Slackアプリが正しくインストールされているか確認
- チャンネルまたはDMが正しく選択されているか確認

## 対処法

### メール通知が届かない場合

1. 設定ページの「通知設定」で、メールアドレスを確認
2. 迷惑メールフォルダを確認
3. メールアドレスを変更して再試行

### Slack通知が届かない場合

1. 設定ページの「Slack連携」で、連携状態を確認
2. Slackアプリを再インストール
3. チャンネルまたはDMを再選択
