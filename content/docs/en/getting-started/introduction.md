# Introduction

Welcome to ReRank AI!

ReRank AI is an SEO automation tool that integrates with Google Search Console to automatically detect ranking drops and propose improvement suggestions using AI.

## What You Can Do with ReRank AI

- **Automatic Rank Monitoring**: Automatically monitor article rankings by integrating with Google Search Console
- **Automatic Rank Drop Detection**: Receive immediate notifications when rankings drop
- **AI Improvement Suggestions**: Automatically generate specific improvement suggestions based on competitor analysis
- **Automatic Rewrite Draft Generation**: Present items to improve in bullet points

## Key Features

### 1. Manual Scan

No GSC integration required. Simply enter a URL and keywords to instantly display current rankings and improvement suggestions.

### 2. Automatic Trigger

After GSC integration, automatically detects ranking drops and performs analysis and notifications.

### 3. Difference Extraction from Competitors

Identifies differences with top-ranking sites and specifically suggests items to add.
