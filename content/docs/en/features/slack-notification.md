# Slack Notification Settings

ReRank AI can send notifications to Slack channels or DMs, which is convenient for team sharing.

## Integration Steps

1. Click "Slack Integration" on the settings page
2. Click "Install Slack App"
3. Select the workspace where you want to receive notifications
4. Select the channel or DM where you want to receive notifications

## Notification Types

- **Channel Notifications**: Send notifications to the specified channel
- **DM Notifications**: Send notifications to the specified user's DM

## Notification Content

Slack notifications include the following information:

- URL of the article with dropped rankings
- Keywords with dropped rankings
- Summary of improvement suggestions
- Link to detailed analysis results

## Important Notes

- Installing the Slack app is required to use Slack notifications
- Channel posting permissions are required to use channel notifications
