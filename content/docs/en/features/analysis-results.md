# Viewing Analysis Results

This guide explains how to view analysis results generated by ReRank AI.

## How to View Analysis Results

1. Check notifications on the dashboard
2. Click "View Improvement Suggestions"
3. View details on the analysis results page

## Information Included in Analysis Results

### 1. Current Ranking

The current search ranking for the specified keywords is displayed.

### 2. Ranking Trends

A graph displays ranking trends over the past 7 days.

### 3. Improvement Suggestions

AI-analyzed improvement suggestions are displayed in bullet points.

### 4. Competitor Site Analysis

Analysis results of top-ranking competitor sites are displayed.

## How to Use Improvement Suggestions

1. **Prioritize**: Address high-priority improvement suggestions first
2. **Improve Gradually**: Don't improve everything at once; improve gradually
3. **Measure Effectiveness**: Check ranking changes after improvements
