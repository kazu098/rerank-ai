# Frequently Asked Questions

Here are frequently asked questions about ReRank AI.

## Pricing

**Q: How many articles can I monitor on the free plan?**
A: The free plan allows you to monitor up to 3 articles.

**Q: What are the prices for paid plans?**
A: The Standard plan is $29/month and the Pro plan is $99/month.

**Q: Can I cancel anytime?**
A: Yes, you can cancel at any time. You can continue using the service until the end of the current billing period.

## Features

**Q: Is GSC integration required?**
A: GSC integration is required for automatic monitoring. It is not required for manual scanning.

**Q: How often will I receive notifications?**
A: You will receive notifications immediately when rankings drop below your set threshold.

**Q: Can I monitor multiple sites?**
A: Yes, you can monitor multiple sites. The number of sites you can monitor depends on your plan.

## Technical Questions

**Q: Which browsers are supported?**
A: The latest versions of Chrome, Firefox, Safari, and Edge are supported.

**Q: Is there a mobile app?**
A: Currently, there is no mobile app, but you can use the service from mobile browsers.

**Q: Is an API available?**
A: Currently, no API is available. We are considering providing one in the future.

## Other

**Q: Is support available in English?**
A: Yes, we provide support in English.

**Q: Is data backed up?**
A: Yes, data is automatically backed up.
