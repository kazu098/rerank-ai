# Notification Not Received

This guide explains how to handle when notifications are not received.

## Things to Check

### 1. Notification Settings

- Check if notifications are enabled in "Notification Settings" on the settings page
- Check if email or Slack notifications are enabled

### 2. Notification Conditions

- Check if the rank drop threshold is set appropriately
- Check if the comparison period is set appropriately

### 3. Email Address

- Check if the registered email address is correct
- Check your spam folder

### 4. Slack Integration

- Check if the Slack app is properly installed
- Check if the channel or DM is correctly selected

## Solutions

### If Email Notifications Are Not Received

1. Check the email address in "Notification Settings" on the settings page
2. Check your spam folder
3. Change the email address and try again

### If Slack Notifications Are Not Received

1. Check the integration status in "Slack Integration" on the settings page
2. Reinstall the Slack app
3. Reselect the channel or DM
