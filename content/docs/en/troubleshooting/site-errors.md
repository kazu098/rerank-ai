# Site Registration Errors

This guide explains how to handle errors that occur when registering sites.

## Common Errors

### Site Not Displayed

**Cause**: Site is not properly registered in GSC

**Solution**:
1. Verify that the site is properly registered in Google Search Console
2. Verify that site ownership verification is complete
3. Try with a different Google account

### Access Permission Error

**Cause**: No access permission to GSC

**Solution**:
1. Check access permissions to the site in Google Search Console
2. Verify that you are added as the site owner or user
3. Try with a different Google account

### Integration Error

**Cause**: GSC integration was not completed correctly

**Solution**:
1. Check the integration status in "GSC Integration" on the settings page
2. Click "Re-authenticate" and try re-authentication
3. Clear browser cookies and try again
